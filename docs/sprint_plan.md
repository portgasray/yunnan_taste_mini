## 注意事项

1. 每个组件开发需遵循以下步骤:
   - 组件设计与规划
   - 组件实现
   - 主题集成
   - 单元测试
   - 文档编写

2. 每日结束时进行简短的团队同步，讨论:
   - 当天完成的工作
   - 遇到的问题
   - 次日计划

3. 生物发光森林主题的关键视觉元素:
   - 深色背景与发光元素对比
   - 渐变与光晕效果
   - 流动的动画效果
   - 有机的形状与自然元素

4. 性能优化重点:
   - 组件按需加载
   - 图片资源优化
   - 动画性能优化
   - 小程序包大小控制
